节奏

Cadence 规则

每 5–7 页内文,眼睛需要一个视觉重置 —— 章节分隔、stat-anchor 或 pull-quote 都行。没有重置,读者会迷路。Cadence 是"成书"和"打字稿"之间的分水岭。

07 / 08

量化限制

密度限值

任何一条超标,这份文档就违反了 Quire。这些限值防的是 AI 最常见的失败:把每一页都当成最重要的那一页。

元素 上限 原因
Callout(Exercise / Note / Think) 每对开页 ≤ 2 个 3 个以上,页面看起来像一节带批注的课,不像书的一章。
Pull-quote 页 每章 ≤ 1 个 Pull-quote 是标点。一章里两个就变装饰。
Stat-anchor 页 每份文档 ≤ 2 个 数字是锚。过多会让文档退化成信息图。
每页辅色面积 凭判断 5% 上限已退役。原则仍在:单一辅色要让人感到稀缺。
Inline .hl 高亮 每页 ≤ 2 处 什么都高亮 = 什么都没高亮。
视觉重置(cadence) 每 5–7 页内文 超过 7 页,读者在长文档里就会失去方位。

08 / 08

硬规则

10 条不变式

Quire 不会破的规则。每一条都有真实的代价 —— 想清楚再覆盖。

  1. 冷调画布 #f6f8fb,绝不纯白,绝不暖奶油色。
  2. 天蓝 #3a82c4 是唯一发色 —— 没有第二色。
  3. 全部灰为冷调(B ≥ G ≥ R)。禁止暖黄褐灰。
  4. 一个衬线字体扛全部层级。Sans 只用在 UI 边角。
  5. 只有两个字重:400 / 500。不加粗,正文不用斜体。
  6. 字号机制:标题收紧,eyebrow 放开,正文为零。
  7. 章节分隔用 --accent-tint,绝不用满辅色。
  8. 无阴影、无渐变、无模糊。border-radius ≤ 3pt。
  9. 无个人品牌饰件 —— 无头像、无徽章、无签名框。
  10. 每个 callout 必须有 label;没有 label 的 blockquote 会沦为装饰。
